Semantica

Aprendi que semantica é uma parte que ajuda o navegador entender o que de fato aqui é , por exemplo: Antes , usavamos a tag div pra tudo, mas hoje em dia temos tags mais especificas como header, footer, main, section, article, entre outras. Essas tags ajudam a organizar o conteudo de forma mais clara e compreensivel para o navegador e para os motores de busca.

Assesibilidade

A acessibilidade é uma parte muito importante em um site, pois existem pessoas com deficiência visual que utilizam softwares leitores de tela para acessar o conteúdo. Construir um site acessível ajuda essas pessoas a consumirem o conteúdo de forma eficiente e inclusiva. Algumas tags que podemos usar é a tag Alt para imagens, que descreve o conteúdo da imagem para os leitores de tela. Além disso, podemos usar a tag aria-label para fornecer informações adicionais sobre elementos interativos. ou adicinado a tag Title

SEO

Aprendemos que SEO (Search Engine Optimization) é um conjunto de práticas que ajudam a melhorar a visibilidade de um site nos motores de busca. Isso inclui o uso adequado de tags HTML, como title , meta description , e a estruturação correta do conteúdo com cabeçalhos ( h1, h2, h3 ). Além disso, a escolha de palavras-chave relevantes e a otimização de imagens também são fundamentais para uma boa estratégia de SEO.

Cores

As cores desempenham um papel crucial no design de um site, pois influenciam a experiência do usuário e a percepção da marca. É importante escolher uma paleta de cores que seja harmoniosa e que transmita a mensagem desejada. Além disso, devemos considerar o contraste entre as cores para garantir a legibilidade do texto e a acessibilidade para pessoas com deficiência visual.

Figure e Imagem

As tags figure e img são usadas para incorporar imagens em uma página HTML. A tag figure é usada para agrupar uma imagem com sua legenda, enquanto a tag img é usada para exibir a imagem propriamente dita. É importante sempre incluir o atributo alt na tag img para descrever o conteúdo da imagem, garantindo acessibilidade para usuários com deficiência visual.

A tag blockquote é usada para destacar citações de outras fontes, enquanto a tag summary é usada para fornecer um resumo de um conteúdo mais extenso, como em listas ou seções de informações.

Tabela e Formulário

As tabelas são usadas para organizar dados em linhas e colunas, facilitando a visualização de informações estruturadas. A tag table é usada para criar uma tabela, enquanto as tags tr , th e td são usadas para definir linhas, cabeçalhos e células de dados, respectivamente.

Os formulários são usados para coletar informações dos usuários. A tag form é usada para criar um formulário, enquanto as tags input , label , textarea , entre outras, são usadas para definir os campos de entrada e rótulos.
